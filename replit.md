# نظام إدارة مدرسة اقرأ الخاصة

## نظرة عامة

نظام شامل لإدارة المدرسة مبني كصفحة HTML واحدة (`iqraa-school/index.html`). يشمل موقعاً عاماً للمدرسة ولوحة تحكم إدارية متكاملة.

## بنية المشروع

- **`iqraa-school/index.html`** - الملف الرئيسي الوحيد (4250+ سطر)
- **`.replit`** - تكوين بيئة Nix
- **`replit.nix`** - تعريف الحزم

## التشغيل

```
python3 -m http.server 5000 --directory iqraa-school
```
الموقع يعمل على المنفذ 5000.

## التقنيات المستخدمة

- **React 18** عبر CDN (UMD)
- **Babel Standalone** لتحويل JSX في المتصفح
- **Tailwind CSS** عبر CDN
- **خطوط Cairo/Tajawal** عبر Google Fonts
- **localStorage** لحفظ البيانات (مشفرة بـ XOR)

## بيانات التخزين

- **مفتاح localStorage**: `iqraaV3`
- **التشفير**: XOR cipher
- **بيانات افتراضية**: في ثابت `DEFAULT_DATA`

## الدخول للوحة الإدارة

- **اسم المستخدم**: `admin`
- **كلمة المرور**: `admin2026`
- **الوصول**: زر "إدارة" في شريط التصفح

## وحدات النظام

### الموقع العام
- الصفحة الرئيسية مع سلايدر
- صفحة عن المدرسة
- صفحة الأخبار والفعاليات
- المعرض والمشاركات
- جدار الشرف
- صفحة الخريجين
- دعم التعلم
- بوابة الأولياء والطلاب
- صفحة التوظيف والوظائف
- التواصل الاجتماعي

### لوحة التحكم الإدارية (7 مجموعات)

1. **الرئيسية**: لوحة المعلومات، الإعلانات، البانر المنبثق
2. **إدارة الموارد البشرية**: الموظفون، الحضور/الغياب، الطلاب، الحافلات
3. **بوابات الأهالي والطلاب**: بوابة الأولياء، آراء الأولياء، الخريجون
4. **المحتوى والأخبار**: الأخبار، المشاركات، المقالات، المعرض، الفيديوهات، السلايدر
5. **المدرسة والطاقم**: طاقم المدرسة، الفرق، الصور، مقالات المعلمين
6. **التقويم والتوظيف**: التقويم، الوظائف، طلبات التوظيف
7. **الإعدادات والنظام**: تخصيص التصميم، اللوجو، التواصل الاجتماعي، الأمان، الإعدادات

## المكونات الرئيسية

| المكوّن | الوصف |

| `ADash` | لوحة معلومات شاملة مع إحصائيات HR والحضور والمحتوى |
