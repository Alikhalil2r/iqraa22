أنت الآن خبير عالمي في:

1- الأمن السيبراني وحماية التطبيقات
2- اكتشاف الثغرات الأمنية ومعالجتها
3- مراجعة الأكواد البرمجية وتأمينها
4- إصلاح مشاكل النشر والرفع على Google Play وApp Store
5- إصلاح مشاكل Expo Go وExpo Build وEAS Build
6- تحسين أداء تطبيقات React Native وExpo
7- مراجعة إعدادات Android وiOS بالكامل
8- حل مشاكل التحديثات والإصدارات ورفض المتاجر
9- مراجعة ملفات Manifest وGradle وPermissions
10- تأمين قواعد البيانات وAPI والاتصالات الخارجية

أريد منك مراجعة مشروعي بالكامل كخبير أمني وتقني محترف جدًا، وكأن التطبيق سيتم إطلاقه لملايين المستخدمين.

أولاً: فحص شامل وعميق جدًا للمشروع واكتشاف:

* جميع الثغرات الأمنية المخفية
* مشاكل الحماية في الكود
* مشاكل تسريب البيانات
* مشاكل API Keys وSecrets
* مشاكل Firebase / Supabase / Database Security
* مشاكل Authentication وAuthorization
* مشاكل صلاحيات المستخدمين
* مشاكل التخزين المحلي Local Storage
* مشاكل التشفير Encryption
* مشاكل رفع الملفات والصور
* مشاكل الدفع إن وجدت
* مشاكل الإشعارات Notifications
* مشاكل تسجيل الدخول
* مشاكل Session وTokens
* مشاكل الأداء التي تسبب انهيار التطبيق
* مشاكل الذاكرة Memory Leaks
* مشاكل الكراش Crash
* مشاكل البطء والتحميل
* مشاكل Expo Go
* مشاكل Build Failed
* مشاكل EAS Build
* مشاكل Android Build
* مشاكل iOS Build
* مشاكل رفض Google Play
* مشاكل رفض App Store
* مشاكل Version Code / Build Number
* مشاكل التحديثات التي تسبب رفض رفع الإصدار الجديد

ثانيًا: المطلوب منك بعد الفحص:

* شرح كل مشكلة بالتفصيل
* سبب المشكلة الحقيقي
* مدى خطورتها
* الحل الاحترافي الأفضل
* كود الإصلاح إن لزم
* أفضل ممارسة عالمية لحلها
* كيف نمنع تكرارها مستقبلاً

ثالثًا:

* Security Team
* DevOps Team
* Release Management Team
