الخطوات المقترحة للتنفيذ
# 1. إنشاء المشروع
npx create-next-app@latest nexgen-website --typescript --tailwind --app
# 2. تثبيت المكتبات
npm install framer-motion lucide-react @tailwindcss/typography
# 3. إضافة خط عربي في layout.tsx
import { Tajawal } from 'next/font/google'
# 4. تفعيل RTL في tailwind.config.ts
plugins: [require('tailwindcss-rtl')]
# 5. إنشاء الـ components
/components: Header, Footer, HeroSection, ServiceCard, ProjectCard, TestimonialCard, ContactForm, ClientPortal
# 6. بناء الصفحات
/app: page.tsx, services/page.tsx, portfolio/page.tsx, about/page.tsx, contact/page.tsx, client-portal/page.tsx

ملاحظات مهمة
RTL أولاً: كل النصوص والعناصر تعمل من اليمين لليسار — استخدم dir="rtl" في <html>
الأداء: استخدم next/image لكل الصور، lazy loading افتراضي
الأمان: نموذج التواصل محمي بـ reCAPTCHA v3 أو Turnstile من Cloudflare
الإمكانية: WCAG 2.1 AA — aria-labels بالعربية، contrast ratio مناسب
التحليلات: اربط Google Analytics 4 أو Plausible (مع احترام الخصوصية)
البريد الإلكتروني: استخدم Resend أو EmailJS لإرسال نماذج التواصل
